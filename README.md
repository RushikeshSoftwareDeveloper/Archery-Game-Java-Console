# 🎯 Archery Game (Java Console)

## 📌 Description
A simple console-based Archery Game developed using Core Java.  
The player controls the angle and power of the arrow to hit a target placed at a fixed distance.

## 🎮 Features
- Adjustable angle and power input
- Target hit or miss detection
- Basic projectile motion logic
- Interactive gameplay

## 🛠 Concepts Used
✔ Math Logic (Angle & Distance)  
✔ Conditional Statements  
✔ Loops  
✔ User Input (Scanner)  
✔ Basic Physics Simulation  

## 🎯 Game Objective
Hit the target by choosing the correct angle and power.

## ▶ How to Run
1. Open project in Eclipse / IntelliJ
2. Run the main class
